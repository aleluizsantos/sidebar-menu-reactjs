# Sidebar

Projeto desenvolvido em react criação de sidebar em reactjs

## Installation

Install my-project with npm

```bash
  git clone https://github.com/aleluizsantos/sidebar-menu-reactjs.git
  
  npm install my-project
  cd my-project

  or 

  yarn 
```
